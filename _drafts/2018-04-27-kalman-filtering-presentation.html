Kalman Filtering
What is a Kalman Filter and What Can It Do?
A Kalman filter is a recursive, optimal, linear estimator.
It is recursive so that new measurements can be processed as
they arrive. (instead of batch processing where all data must be
present).

Kalman Filtering
Optimal in what sense?
If all noise is Gaussian, the Kalman filter minimizes the mean square error of
the estimated parameters.
It is the MMSE estimator (and also the MAP estimator under the Kalman
assumptions).

http://biorobotics.ri.cmu.edu/papers/sbp_papers/integrated3/kleeman_kalman_basics.pdf

UKF: Idea

Pick points around the current mean, spread

Run each point through the nonlinear function
●
Use where they end up to approximate the new

Particle Filtering
●
A type of Sequential Monte Carlo method
●
Idea: discretize the pdf into a pmf.
–
Get a large number of weighted points
–
Propagate each through the nonlinear dynamics
–
When a measurement comes, give ones that have a higher
likelihood a larger weight
–
If particles are too low, kill them, and resample based on the
current weights
–
Good solutions will emerge, bad estimates of the pdf will die
out
